搭建个人博客hexo

搭建

1.安装node.js

2.(npm 安装cnpm)

3.使用cnpm/npm 安装hexo 博客框架 ： cnpm install -g hexo-cli

4.新建文件夹（放置博客内容文件）

5.在文件夹下 hexo init

6.启动hexo： hexo s

  • 建立新博客 $ hexo new “My New Post”

  • 运行服务器 $ hexo server

  • 生成静态文件 $ hexo generate

  • 部署到远程站点 $ hexo deploy
