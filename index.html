其他问题

1 .HV主机服务开启关闭方法

  • 快捷键win+x,使用管理员权限打开Windows Powershell
  • 关闭：bcdedit /set hypervisorlaunchtype off
  • 开启: bcdedit /set hypervisorlaunchtype auto

2.idea编译问题

Java8中增加了通过反射获取方法参数名的功能，并且这些功能在新版的Springboot里边有用到，但是idea比较旧的版本编译时不会使用 javac -parameters 命令来编译。因此在旧版本中需要在 idea中找到File->Settings->java Compiler中的Additional command line parameters添加-parameters参数即可 。

搭建个人博客hexo

发表于 2020-03-29

搭建

1.安装node.js

2.(npm 安装cnpm)

3.使用cnpm/npm 安装hexo 博客框架 ： cnpm install -g hexo-cli

4.新建文件夹（放置博客内容文件）

5.在文件夹下 hexo init

6.启动hexo： hexo s

  • 建立新博客 $ hexo new “My New Post”

  • 运行服务器 $ hexo server

  • 生成静态文件 $ hexo generate

  • 部署到远程站点 $ hexo deploy

7.部署到github 上，new repository，命名必须是xxx.github.io

8.在文件夹下安装插件 cnpm install –save hexo-deployer-git

9.修改配置文件
